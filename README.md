# Projeto Vitória SQL

Projeto desenvolvido para praticar SQL utilizando dados do Esporte Clube Vitória.

## Objetivo

Realizar consultas SQL para análise de desempenho do clube, utilizando funções agregadas, filtros e outras operações de banco de dados.

## Tecnologias Utilizadas

- MySQL
- MySQL Workbench
- GitHub

## Consultas Realizadas

- Média de gols por partida
- Soma de gols marcados
- Menor quantidade de gols marcados
- Consultas com WHERE
- Consultas com funções agregadas
